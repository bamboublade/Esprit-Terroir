Suppression des Données Personnelles

Conformément au RGPD et aux politiques de confidentialité de nos partenaires (Facebook, Instagram), vous avez le droit de demander la suppression de toutes les données personnelles vous concernant traitées par Esprit Terroir.

Comment demander la suppression ?

Si vous souhaitez que nous supprimions vos informations de notre base de données ou si vous souhaitez révoquer l'accès à vos données sociales via notre flux Instagram intégré, veuillez suivre les étapes suivantes :

Option 1 : Par Email (Recommandé)

Envoyez une demande formelle à l'adresse suivante :
contact@esprit-terroir.com

Précisez dans l'objet : "Demande de suppression de données - RGPD". Nous traiterons votre demande sous 30 jours maximum.

Option 2 : Via Facebook / Instagram

Si vous avez interagi avec notre application Instagram via ce site et que vous souhaitez révoquer l'accès aux données :

  • Allez dans les paramètres de votre compte Facebook/Instagram.
  • Accédez à "Paramètres et confidentialité" > "Applications et sites web".
  • Recherchez "Esprit Terroir" et cliquez sur "Supprimer".

Ce que nous supprimons

Lors d'une demande de suppression, nous effaçons de nos serveurs :

  • Vos coordonnées de contact (emails, numéros de téléphone) envoyées via formulaire ou contact direct.
  • Toute trace d'historique de communication marketing si applicable.
  • Les éventuels jetons d'accès sociaux liés à votre profil.

Pour toute question supplémentaire sur la gestion de vos données, vous pouvez également consulter notre Politique de Confidentialité.
